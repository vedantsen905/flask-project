About My To-Do List

Welcome to my To-Do List application! This web application allows you to manage your tasks easily and efficiently. You can add, view, update, and delete tasks as needed.

Features:

  • Add new tasks with a title and description
  • Edit existing tasks
  • Delete completed or unwanted tasks
  • View all tasks in a simple, user-friendly interface

This project was built using Flask for the backend, and it uses SQLite as the database to store your tasks.
